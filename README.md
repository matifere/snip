# snip
Un simple gestor de snippets utilizando supabase

## instalar
Aun no es instalable, proximamente se agregara una opcion para descargarlo en tu plataforma de preferencia.
Actualmente la unica forma de usar la app es compilandola por cuenta propia usando flutter y creando tu propia base de datos en supabase. Para esto no hay que olvidar crear un *.env* donde se carguen los datos privados de la DB
